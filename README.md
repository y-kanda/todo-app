Todo App（React + TypeScript）

■ 概要

タスク管理ができるシンプルなTodoアプリです。

■ URL

https://codesandbox.io/p/sandbox/react-typescript-forked-nnwxt6

■ 機能

* タスク追加
* 完了 / 未完了切り替え
* 削除機能
* フィルター機能
* localStorage保存

■ 使用技術

* React
* TypeScript
* Tailwind CSS

■ 工夫した点

* 親で状態管理、子で表示を分離
* map / filterで状態更新
* UX改善（Enter追加・空状態表示）
